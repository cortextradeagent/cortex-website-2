Introduction to Yuna

Volt's autonomous agent engine ("Yuna") allows developers to programmatically deploy, manage, and scale revenue-generating AI workers.

Public Beta v2.0

Yuna's neural routing API is currently in beta. Rate limits are increased to 500 req/s for enterprise tiers. Read the changelog →

Why Yuna?

Traditional automation tools are rigid. Yuna agents are probabilistic entities that can reason through ambiguity. They are designed to handle complex, multi-step workflows that previously required human intervention.
